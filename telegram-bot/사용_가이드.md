# 📱 매니아그룹 텔레그램 봇 사용 가이드

## 🎯 개요

대표님이 핸드폰으로 편하게 업무 자료를 관리할 수 있도록 만든 텔레그램 봇입니다.
카카오톡처럼 편하게 사용하면서, 모든 자료가 자동으로 웹사이트에 정리됩니다!

---

## ⚙️ 초기 설정 (1회만)

### 1. 텔레그램 봇 만들기

1. 텔레그램 앱 열기
2. 검색창에서 `@BotFather` 검색
3. 대화 시작 후 아래 명령어 순서대로 입력:

```
/newbot
매니아그룹봇
maniagroup_admin_bot
```

4. 받은 토큰 저장 (예: 123456789:ABCdefG...)

### 2. 그룹챗 만들기

1. 텔레그램에서 새 그룹 만들기
2. 그룹명: "매니아그룹 업무실"
3. 방금 만든 봇을 그룹에 추가
4. 봇을 관리자로 승격

### 3. 설정 파일 만들기

프로젝트 폴더에서 `.env.local` 파일 생성:

```env
TELEGRAM_BOT_TOKEN=여기에_봇_토큰_붙여넣기
TELEGRAM_GROUP_CHAT_ID=여기에_그룹_ID_붙여넣기
```

---

## 🚀 봇 시작하기

### 방법 1: 자동 시작 (권장)

```bash
npm run bot
```

### 방법 2: 수동 시작

```bash
cd telegram-bot
node bot.js
```

봇이 시작되면 이런 메시지가 나타납니다:
```
🤖 텔레그램 봇이 시작되었습니다!
✅ 봇이 준비되었습니다. 메시지를 기다리는 중...
```

## 📱 핸드폰에서 사용하기

### 기본 명령어

그룹챗에서 이렇게 입력하세요:

| 명령어 | 설명 |
|--------|------|
| `/start` | 봇 시작 및 환영 메시지 |
| `/업로드` | 파일 업로드 가이드 |
| `/목록` | 최근 업로드된 파일 10개 |
| `/프로젝트` | 진행 중인 프로젝트 목록 |
| `/파트너` | 협력 파트너 목록 |
| `/제품` | 제품 카탈로그 |
| `/도움말` | 자세한 사용법 |

## 📤 파일 업로드 방법

### 방법 1: 일반 파일 (가장 많이 사용)

1. 그룹챗 열기
2. 📎 클립 아이콘 누르기
3. 파일 선택
4. (선택) 설명 추가
5. 전송!

**예시:**
```
📎 파일: 신규_제품_카탈로그.pdf
💬 설명: 2026년 신제품 라인업
```

→ 자동으로 웹사이트에 업로드됨!

### 방법 2: 사진/이미지

1. 사진 촬영 또는 갤러리에서 선택
2. 전송

→ 자동으로 이미지 폴더에 저장됨!

### 방법 3: 여러 파일 한번에

1. 파일 여러 개 선택
2. 한번에 전송

→ 모두 자동 처리됨!

## 💡 활용 팁

### 1. 파일 정리 팁

- 파일 이름에 날짜 포함: `2026-02-17_제안서.pdf`
- 설명 꼭 추가하기: "비티진 마케팅 제안서"
- 중요한 파일은 ⭐ 이모지 붙이기

### 2. 빠른 작업

- 급한 파일은 그냥 업로드 (나중에 웹사이트에서 정리)
- 간단한 메모도 텔레그램에 남기기
- 대표님이 승인할 건 📝 이모지 붙이기

### 3. 팀 협업

- 중요한 결정사항은 텔레그램에 기록
- 파일 공유는 텔레그램으로
- 회의록도 텔레그램에 올리기

## 🔧 문제 해결

### 봇이 응답하지 않아요

1. 봇이 실행 중인지 확인
   ```bash
   npm run bot
   ```

2. 그룹에 봇이 있는지 확인
   - 그룹 멤버 목록에서 확인

3. 봇의 권한 확인
   - 관리자 권한 있어야 함

### 파일이 업로드되지 않아요

1. 파일 크기 확인 (20MB 이하)
2. 파일 형식 확인 (PDF, Word, Excel, 이미지 등)
3. 인터넷 연결 확인

### 알림이 오지 않아요

1. 그룹 ID가 올바른지 확인
2. 봇이 그룹에 있는지 확인
3. 봇 설정 확인

## 🌐 웹사이트 연동

### 업로드된 파일 확인

1. 웹사이트 접속: http://localhost:3000
2. 로그인 (admin@maniagroup.com / admin123)
3. 문서 관리 페이지에서 확인

### 제품/파트너 확인

- 제품 관리: `/admin/products`
- 파트너 관리: `/admin/partners`
- 프로젝트 관리: `/admin/projects`

## 📊 자동 분류 시스템

업로드한 파일은 자동으로 분류됩니다:

### 파일명 기반 분류

- `제안서` → 마케팅 카테고리
- `계약서` → 법무 카테고리
- `제품` → 제품 카테고리
- `파트너` → 파트너 카테고리

### 수동 재분류

웹사이트 설정 페이지에서:
1. "수동 분류 실행" 버튼 클릭
2. 자동으로 모든 파일 재분류

## 🎯 실전 시나리오

### 시나리오 1: 새 제품 자료 받음

```
1. 이메일로 받은 제품 카탈로그 PDF
2. 텔레그램 그룹에 업로드
3. 설명: "2026 신제품 카탈로그"
4. 자동으로 제품 카테고리에 저장됨
5. 웹사이트에서 제품 정보 확인
```

### 시나리오 2: 회의 중 파트너 정보 추가

```
1. 텔레그램 그룹 열기
2. 입력: "파트너: 홍길동마케팅 / 홍길동 / 010-1234-5678"
3. 즉시 등록 완료
4. 나중에 웹사이트에서 상세 정보 추가
```

### 시나리오 3: 외근 중 사진 공유

```
1. 현장에서 사진 촬영
2. 텔레그램 그룹에 전송
3. 설명: "매장 현황 2026-02-17"
4. 자동으로 저장됨
```

## ⚠️ 주의사항

### 보안

- ❌ 봇 토큰 절대 공개 금지
- ❌ 그룹 초대 링크 공유 금지
- ✅ 필요한 인원만 그룹에 추가

### 파일 관리

- 중요한 파일은 백업 필수
- 대용량 파일(20MB 이상)은 웹사이트에서 직접 업로드
- 민감한 정보는 암호화된 파일로

### 운영

- 봇은 24시간 실행 권장
- 서버 재시작 시 봇도 다시 실행
- 정기적으로 로그 확인
