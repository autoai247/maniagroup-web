# 📱 초간단 사용법 - 3초 만에 파일 찾기!

## 🎯 가장 쉬운 방법 (추천!)

### 1단계: 키워드만 입력
```
제품
```

### 2단계: 버튼 클릭
```
[📄 제품카탈로그.pdf]
[📄 제품리스트.xlsx]
[📄 신제품소개.pptx]
```

### 3단계: 완료!
→ 파일이 바로 다운로드됨! ✅

---

## 🔥 자주 사용하는 키워드

그냥 이것만 입력하세요:

| 키워드 | 찾는 것 |
|--------|---------|
| `제품` | 제품 관련 모든 파일 |
| `파트너` | 파트너사 자료 |
| `계약` | 계약서 |
| `제안` | 제안서 |
| `보고서` | 각종 보고서 |
| `카탈로그` | 카탈로그 |
| `이미지` | 사진/이미지 파일 |
| `최근` | 최근 업로드 파일 |

---

## 💡 실전 예시

### 예시 1: 제품 카탈로그 찾기
```
👤 대표: 제품

🤖 봇: 🔍 "제품" 관련 파일:
     [📄 제품카탈로그_2026.pdf]
     [📄 신제품_리스트.xlsx]
     [📄 제품_소개서.pptx]

👤 대표: [첫 번째 버튼 클릭]

🤖 봇: [제품카탈로그_2026.pdf 전송]
     ✅ 다운로드 완료!
```

### 예시 2: 계약서 찾기
```
👤 대표: 계약

🤖 봇: 🔍 "계약" 관련 파일:
     [📄 파트너_계약서.pdf]
     [📄 계약서_템플릿.docx]

👤 대표: [원하는 파일 클릭]

🤖 봇: [파일 전송] ✅
```

### 예시 3: 최근 파일 보기
```
👤 대표: /목록

🤖 봇: 📋 최근 업로드된 파일:
     [📄 오늘 업로드한 파일1]
     [📄 오늘 업로드한 파일2]
     ...

👤 대표: [클릭]

🤖 봇: [파일 전송] ✅
```

---

## 🎨 더 정확하게 찾기

정확한 파일명을 알 때:

```
/찾기 카탈로그
```

→ "카탈로그"가 포함된 파일만 검색!

---

## 📤 파일 올리기

### 방법 1: 핸드폰에서
```
1. 클립(📎) 아이콘 클릭
2. 파일 선택
3. 전송!
```

### 방법 2: 사진 올리기
```
1. 카메라 아이콘 클릭
2. 사진 촬영 또는 갤러리 선택
3. 전송!
```

→ 자동으로 분류되어 저장됨!

---

## ⚡ 초고속 사용 팁

### 자주 찾는 파일
```
제품 → [즐겨찾기 파일들]
파트너 → [자주 쓰는 파트너 자료]
계약 → [계약서 모음]
```

### 빠른 업로드
```
파일 첨부 → 설명 추가 → 전송
예: "신제품 카탈로그 - 2026년 1분기"
```

### 여러 파일 한번에
```
파일 3개 선택 → 한번에 전송
→ 모두 자동 처리됨!
```

## 🆘 문제 해결

### Q: 파일이 안 찾아져요
```
A: /목록 으로 전체 파일 확인
   파일명을 정확히 기억한다면
   /찾기 정확한파일명
```

### Q: 버튼이 안 눌러져요
```
A: 봇이 실행 중인지 확인
   서버 재시작 필요할 수도 있음
```

### Q: 파일이 안 올라가요
```
A: 파일 크기 확인 (20MB 이하)
   파일 형식 확인
   인터넷 연결 확인
```
